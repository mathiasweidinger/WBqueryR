## TL;DR

A "brick and mortar" R-package that makes it easy to query the World Bank's Microdata Library for variables from within R. Its main function `WBqueryR::WBquery()` takes user-defined search parameters and a list of keywords as input, downloads codebooks that meet the search criteria, and queries the variable labels in them for the presence of the keywords.
